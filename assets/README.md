# Assets

These are placeholder images. Replace them with your real app graphics before publishing.

## Required Files:

| File | Size | Purpose |
|------|------|---------|
| icon.png | 1024x1024 | App icon for all platforms |
| adaptive-icon.png | 1024x1024 | Android adaptive icon foreground |
| splash.png | 1242x2436 | Splash screen background |
| favicon.png | 48x48 | Web favicon |

## Easy Way to Generate Icons:

1. Go to https://icon.kitchen/
2. Upload your logo/image
3. Download the generated assets
4. Replace the placeholder files in this folder

## Note:
The current files are 1x1 transparent PNGs (67 bytes each).
They won't crash the build but won't look good either.
Replace them before sharing with friends!
